Project information

  • NAME: ArtistToget
  • Category: Web app
  • Frontend: React js
  • Backend: PHP(Codigniter)
  • Project date: 01 Jan, 2022 To Present

Project Detailed Description

  • "ArtistToget" is a web application that provides platform for struggling artist and other users to hire artists with different Forte.
  • This platform is aimed to work as a life line for struggling artists who are talented but do not have exposure to the clients who want artists for thier different projects.
  • Users with art projects post thier project details on platform and artists bid for the projects they are interested in.
  • User assigns the project to the artist to get thier work easily by not facing any difficulties. the payment is done using third party application.
  • After payment a contract is generated. Here artist work as a freelancer. also grow thier skill to seen other works.
